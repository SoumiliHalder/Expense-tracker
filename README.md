# Expense Tracker Dashboard

A GUI-based Expense Tracker built using Python, Tkinter, CSV, and Matplotlib.

## Features

- Add Expenses
- Edit Expenses
- Delete Expenses
- Search Expenses
- Monthly Expense Report
- Pie Chart Visualization
- Bar Chart Visualization

## Technologies Used

- Python
- Tkinter
- CSV
- Matplotlib

## How to Run

Run:

```bash
python main.py
